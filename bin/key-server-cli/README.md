# Key-Server Cli
Cli for interacting with the key server

## Usage
```
Usage: key-server-cli [OPTIONS] <COMMAND>

Commands:
  layer         
  layer-idx     
  num-layers    
  add-layer     
  remove-layer  
  switch-layer  
  up-layer      
  down-layer    
  save-layout   
  variables     
  set-variable  
  get-variable  
  help          Print this message or the help of the given subcommand(s)

Options:
  -i, --ipc <IPC>  
  -h, --help       Print help information
```
